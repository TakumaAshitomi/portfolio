# Coaching
## https://damp-reaches-55435.herokuapp.com/

ユーザー同士で得意な物のコーチングを共有するサイトです。

- 大まかな機能
    - ログイン・ゲストログイン機能
        - 管理ユーザー機能
    - wysiwygを用いてのコーチング投稿機能
    - 投稿へのタグ付け機能
    - 投稿タイトル・タグ・ユーザーでの検索機能
    - ユーザーの購読(フォロー)機能
    - フォローしたユーザーの投稿一覧機能

- 開発環境
    - 言語
        - Ruby (2.5.3)
        - Ruby on Rails 5.2.3
    - 環境
        - Docker
        - circleCIで自動テスト/herokuに自動デプロイ
    - DB
        - postgreSQL
